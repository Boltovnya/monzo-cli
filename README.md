# Usage
## Balance

```bash
$ monzo-cli balance

+---------+-------------+
| Balance | Spent today |
+---------+-------------+
| £490    | £10         |
+---------+-------------+
```

## Accounts

```bash
$ monzo-cli accounts

+---------------------+----------------------+
| Description         | Date created         |
+---------------------+----------------------+
| Peter Pan's Account | 2015-11-13T12:17:42Z |
+---------------------+----------------------+
```
